# claude-code-action-demo

Sandbox repository used to validate the **AceData Cloud × Claude Code GitHub Actions** integration end-to-end. The workflow under `.github/workflows/claude.yml` calls Anthropic-compatible APIs through `https://api.acedata.cloud`.

If you want to use Claude Code on your own repositories with AceData Cloud as the backend, see the official guide:
<https://platform.acedata.cloud/documents/claude-code-github-actions-integration>
